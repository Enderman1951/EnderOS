Welcome to EnderOS

<img src="https://i.imgur.com/aNh4xts.png" alt="EnderOS Running">

EnderOS is a live, Arch Linux based Linux Distro for users who need a portable, disposable, and privacy-respecting environment. EnderOS is general-purpose: you can code, edit media, take notes, and work in office apps.

By default, EnderOS does not retain any files or session data. To save work, you must use:

- External drives

- Cloud storage

- NAS or remote storage

Default Desktop Envoirment
- XFCE

SYSTEM REQUIREMENTS (MINIMUM):

- UEFI Enabled Motherboard (with secure boot turned off)
- 64 Bit x86 CPU
- 4GB of RAM
- A USB Flash Drive/Portable SSD with atleast 8GB of storage

Download the ISO from [releases](https://github.com/Enderman1951/EnderOS/releases). You can also use [Archiso](https://wiki.archlinux.org/title/Archiso) to build the EnderOS ISO as of right now. (Releases coming soon)
